# NicerDicer
A weaverdice discord bot.

## Features - Basic
- [ ] Dice Rolls (Format: /roll 3d6+4)
- [ ] Wound Rolls (See: https://docs.google.com/document/d/1qyyXT2FGvWfe8SxF0oLEXbNnuXGAolSXYwHNjErBCbA/edit?usp=sharing, under "Wound Effects, Format: /wound 1m1l poison 1m bash)
- [ ] Augment/Perk Rolls (Format: /augmen, /perk, see: https://docs.google.com/spreadsheets/d/1UK1o9Vb15kYFVkfa9qEh9BBNCB7R6IBIUMD1YSHOHjE/edit?gid=403439930#gid=403439930)
- [ ] Tag System (You know it, you love it, will copy from issei)

## Features - Advanced
- [ ] Initiative / Turn funcitonality (Format: /init or /init +3 for modifiers; /startcombat; /endturn; /endcombat)
- [ ] Draw Cards
